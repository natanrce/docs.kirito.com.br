# Prefácio

Após adquirir conhecimento sobre diversas ferramentas e aprender a instalar cada uma manualmente, existe uma solução que pode facilitar esse processo: o Mobox. Trata-se de uma ferramenta que agiliza e simplifica a montagem do seu setup, poupando seu tempo e esforço.

## Requisitos Mínimos

Antes de prosseguir é importante garantir que o seu ambiente atenda aos requisitos mínimos. Certifique-se de ter instalado o Python 3, Python 2 e o pip. Caso não tenha essas ferramentas instaladas, siga as instruções abaixo:

```bash
sudo apt-get update -y && sudo apt-get upgrade && apt-get install curl python3 python3-pip python2 -y
```

```bash
curl https://bootstrap.pypa.io/pip/2.7/get-pip.py -o get-pip.py ; python2.7 get-pip.py
```
